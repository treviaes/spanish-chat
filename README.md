Spanish chat plugin

## Description ##
This plugin add a chatroom to any page. You must use a shortcode like this:
[chatroom user="user" room="chatroom" width="100%" height="500px"]
You can configure the user name and chatroom but afterthat any user can change the the username or the chat room. Also you can modify the height and width of the shortcode.

## Installation ##
* If you don’t know how to install a plugin for WordPress, [here’s how](http://codex.wordpress.org/Managing_Plugins#Installing_Plugins).

### Requirements ###
* PHP 5.2.4 or greater
* WordPress 3.8 or greater

### Settings ###
After you have activated *Spanish Chat* you must to add the shortcode to any page or post.
The plugin will display a ustom window inside that page or post with your configuration
[chatroom user="user" room="chatroom" width="100%" height="500px"]
user: name of the user
room: name of the chat room like your domain.com
width: The width of the chatroom. The most common option is 100% to adapt to your theme.
height: the height of the chatroom. The most common option is between 500px and 700px. You must check the best height for your design.


## Installation ##
1. Download the plugin
2. Upload the spanishchat folder to your site's
wp-content/plugins directory
3. Activate the plugin in the WordPress Admin Panel
4. Start creating new shortcode
5. View the resulting page
